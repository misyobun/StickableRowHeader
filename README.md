### Step1

Add it in your root build.gradle at the end of repositories:

```
allprojects {
   repositories {
			 ...
   	maven { url 'https://jitpack.io' }
   }
}	
```
### Step2

Add the dependency
```
dependencies {
   implementation 'com.github.misyobun:StickableRowHeader:1.0.1'
}
```
